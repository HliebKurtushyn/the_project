# Проєкт: Онлайн-магазин одягу на Django

## Мета проєкту
Створити функціональний вебмагазин одягу, де користувачі можуть переглядати товари й оформляти замовлення 

## Для чого
- Забезпечити можливість купувати одяг онлайн
- Надати зручний і красивий, сучасний інтерфейс
- Адміністрація може легко керувати товарами та замовленнями через панель адміністратора Django

## Цільова аудиторія
- Люди, які хочуть купувати одяг онлайн.
- Адміністратори магазину, які керують каталогом і акціями.

## Основні функціональні цілі
1. **Акаунти користувачів**
   - Реєстрація та авторизація.
   - Персональний кабінет з історією замовлень.
2. **Оформлення замовлень**
   - Додавання товарів у кошик.
   - Підтвердження замовлення.
3. **Акції та знижки**
   - Відображення знижок на товарах.
   - Керування через адміністративну панель.
4. **Адміністрування**
   - Керування товарами, категоріями, замовленнями через Django Admin.
   - Можливість редагувати ціни.
5. **Інтерфейс**
   - Простий, зрозумілий дизайн для користувачів.
